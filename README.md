# app-prediccion-rendimiento-cafe
Aplicación que usa un modelo CLR el cual permite predecir con los parametros diligenciados por un usuario la productividad en diferentes parcelas de cultivos de café. Junto con el anterior dato usando LLM se entregan recomendaciones agronomicas personalizadas por cultivo.
Se debe asegurar antes de ejecutar la aplicación tener instalado streamlit, verificado lo anterior ejecutar con: streamlit run app_tooltip_v11_groq_gemini.py
